# Main project 배달대

## 개요

프로젝트명 : 메인 프로젝트 - 배달대

개발인원 : 8명 (FE - 3명 / BE - 5명)

개발 기간 : 2023.01.09~2023.02.05

프로젝트 목표 : 협업 능력 증진 ( API 를 활용한 데이터 송수신 / 소통 능력 )

주요 기능 :

회원가입 / 로그인 / 로그아웃 / 개인 정보수정 / 회원탈퇴 /
학교별 상점 검색 / 시간대별 상점 갱신 / 상점별 메뉴, 리뷰, 정보 불러오기 /
메뉴별 옵션 선택 불러오기 / 장바구니 및 주문표 담기 / 결제 및 리뷰작성 /
